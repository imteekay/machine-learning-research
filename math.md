# Mathematics

## Calculus

### Derivatives

Sum of functions

```
h(x) = f(x) + g(x)
h'(x) = f'(x) + g'(x)
```

Subtraction of functions

```
h(x) = f(x) - g(x)
h'(x) = f'(x) - g'(x)
```

Multiplication of functions

```
h(x) = f(x) • g(x)
h'(x) = f'(x) • g(x) + f(x) • g'(x)
```

Division of functions

```
h(x) = f(x) / g(x)
h'(x) = (f'(x) • g(x) - f(x) • g'(x)) / (g(x))²
```
