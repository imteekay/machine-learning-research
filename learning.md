# Learning

## Cognitive Mechanics: Active Retrieval, Automaticity, and Layering

Learning: Transfer data from working memory -> long-term memory

- Deliberate practice and active learning: effortful retrieval of information (rather than passive consumption)
- Continuous problem solving
- Continuous algorithm implementation from scratch to force neural adaptation
- Micro-scaffolding: break down complex concepts into into discrete steps and achieve mastery on each prerequisite
- Automacity: practice skills until they reach a state of automacity (free up working memory)
- Spaced repetition and interleaving: learn concepts reviewed in at distributed interval to prevent decay of long-term memory
