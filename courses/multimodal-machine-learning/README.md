## Introduction

- Modality: something expressed or perceived
  - Raw modalities: speech signal, image
  - Abstract modalities (abstracted from raw modalities): sentiment analysis, object categories
- Multimodal: heterogeneous and interconnected data
  - heterogeneous: different types of data
  - interconnected (connected and interacting): bring these types of data together
